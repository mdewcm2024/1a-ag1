分組專案

1.了解基本 html 語法

HTML 文檔由元素組成，元素由標籤定義。每個標籤都有其自己的含義和用途。例如，<p> 標籤用於創建段落，而 <h1> 標籤用於創建標題。

2.練習創建簡單的html頁面

了解html基本架構，就可以開始練習創建簡單的html頁面。需要有一個文本編輯器和一個Web瀏覽器。

打開編輯器並創建新文件，將以下代碼保存為index.html：

<!DOCTYPE html>

  <title>My First Web Page</title>
</head>
<body>
  <h1>This is my first heading</h1>
  <p>This is a paragraph of text.</p>
</body>
</html>

再打開Web瀏覽器並將文件拖到瀏覽器窗口，就可以看到頁面了。

2.了解如何在頁面導入 Javascript 程式

  • 內聯 JavaScript：將 JavaScript 程式碼直接寫在 HTML 元素的 onclick 等事件屬性中。
  • 內嵌 JavaScript：在 HTML 文件的 <script> 標籤中直接寫 JavaScript 程式碼。
  • 外部 JavaScript：將 JavaScript 程式碼寫在一個單獨的 .js 文件中，然後在 HTML 文件中使用 <script> 標籤引用該文件。

1.內聯 JavaScript

這種方式將 JavaScript 代碼直接嵌入到 HTML 元素中，例如：

<!DOCTYPE html>
<html lang="zh-Hant">
<head>
    <meta charset="UTF-8">
    <title>內聯 JavaScript 範例</title>
</head>
<body>
    <button onclick="alert('按鈕被點擊了')">點我</button>
</body>
</html>

2.內嵌 JavaScript

將 JavaScript 代碼寫在 <script> 標籤內，並將其放在 HTML 文件的 <head> 或 <body> 部分中。例如：

<!DOCTYPE html>
<html lang="zh-Hant">
<head>
    <meta charset="UTF-8">
    <title>內嵌 JavaScript 範例</title>
    <script>
        function showMessage() {
            alert('按鈕被點擊了');
        }
    </script>
</head>
<body>
    <button onclick="showMessage()">點我</button>
</body>
</html>

3.外部 JavaScript

將 JavaScript 代碼寫在一個單獨的 .js 文件中，例如 script.js，然後在 HTML 文件中引用該文件：

function showMessage() {
    alert('按鈕被點擊了');
}

HTML 文件內容：

<!DOCTYPE html>
<html lang="zh-Hant">
<head>
    <meta charset="UTF-8">
    <title>外部 JavaScript 範例</title>
    <script src="script.js"></script>
</head>
<body>
    <button onclick="showMessage()">點我</button>
</body>
</html>

注意事項

  1. 文件位置：確保外部 JavaScript 文件的路徑正確。
  2. 加載順序：若 JavaScript 代碼需要操作 DOM 元素，請確保在 DOM 加載完畢後執行代碼。例如，可以將 <script> 標籤放在 <body> 結束標籤之前，或使用 window.onload 事件。
  3. 調試：使用瀏覽器開發者工具來調試和檢查 JavaScript 代碼。
